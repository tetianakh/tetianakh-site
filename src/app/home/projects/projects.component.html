Vampiric Tutor
vampirictutor.com

A web app which helps you find the best Magic: The Gathering deals on cardmarket.eu

The back end is implemented in Python. It uses Flask as a web app framework, Celery for async tasks, PostgreSQL as a database, and Redis as a cache. The front end is written in Vue.js and uses bootstrap-vue for the styling.

The app is hosted on Heroku.
